# 安装说明

## 环境要求

- LightSNS 站点程序。
- PHP 8 或以上版本。
- 服务器允许 PHP 发起外部 HTTP/HTTPS 请求。
- 推荐开启 LightSNS Worker，以便异步发布事件能及时消费。

## 安装步骤

1. 下载 Release 中的插件压缩包。
2. 解压后确认目录名为：

   ```text
   czzz-pc-page-indexnow-auto-submit
   ```

3. 上传到站点插件目录：

   ```text
   module/pc/page/czzz-pc-page-indexnow-auto-submit
   ```

4. 登录 LightSNS 后台，进入模块管理。
5. 找到 `IndexNow 自动提交` 并启用。
6. 打开插件设置页，保存基础配置。
7. 点击 `维护验证文件`，确保 Key 文件可以通过浏览器公开访问。

## 推荐配置

- `启用自动提交`：开启。
- `监听帖子发布/更新事件`：开启。
- `IndexNow 提交端点`：默认 `https://api.indexnow.org/indexnow`。
- `自动提交去重窗口`：默认 `3600` 秒。
- `历史记录保留天数`：默认 `30` 天。

## 验证安装

1. 在插件设置页进入 `控制台概览`，确认验证状态正常。
2. 在 `手动操作` 中提交一个站内内容 URL。
3. 在 `提交日志` 中查看 HTTP 状态。
4. 发布一篇测试帖子，等待片刻后刷新 `提交日志`，确认出现 `auto` 或 `auto_catchup` 类型记录。

## 常见问题

### 为什么 Bing 后台没有马上显示？

Bing 站长后台不是实时刷新。建议先以插件自己的 `提交日志` 为准。

### HTTP 422 是什么原因？

通常表示提交 URL 与 KeyLocation 验证站点不匹配。请检查：

- Key 文件 URL 是否能公开访问。
- Key 文件内容是否等于当前 Key。
- 提交的 URL 是否属于当前站点域名。
- 站点配置中的 `homeUrl` 是否正确。

### 自动提交没有立刻出现怎么办？

自动提交走异步事件和 Worker，可能有短暂延迟。可以先查看 Worker 是否运行，再用 `补扫最近已发布内容` 做一次补提交。
